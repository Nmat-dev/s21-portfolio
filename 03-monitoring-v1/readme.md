# 🐧 Linux Monitoring v1.0 

> Набор Bash-скриптов для сбора информации о системе, визуализации данных и анализа файловой системы. Проект развивает навыки системного администрирования, работы с утилитами Linux и написания модульных скриптов.

## 🎯 Цель проекта
Освоить написание декомпозированных Bash-скриптов с валидацией ввода, работой с системными вызовами и форматированным выводом данных.

## 🛠 Технологический стек
| Категория | Инструменты |
|-----------|-------------|
| **Язык** | Bash (POSIX-compliant) |
| **Системные утилиты** | `hostname`, `timedatectl`, `whoami`, `uptime`, `ip`, `free`, `df`, `du`, `find`, `md5sum`, `date` |
| **Обработка данных** | `awk`, `grep`, `sed`, `sort`, `head`, `wc`, `bc` |
| **Визуализация** | ANSI escape codes (цветной вывод в терминал) |
| **ОС** | Ubuntu Server 20.04 LTS |
