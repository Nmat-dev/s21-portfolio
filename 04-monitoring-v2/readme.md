# 📊 Linux Monitoring v2.0

> Комплексный проект по мониторингу системы в реальном времени, анализу логов Nginx и визуализации метрик через стек Prometheus + Grafana. Включает написание параметрических Bash-скриптов, генерацию нагрузки и создание кастомного экспортера метрик.

## 🎯 Цель проекта
Освоить инструменты мониторинга и визуализации, научиться анализировать логи веб-серверов, генерировать тестовую нагрузку и интегрировать собственные метрики в Prometheus. Развить навыки декомпозиции Bash-скриптов и работы с инфраструктурным стеком.

## 🛠 Технологический стек
| Категория | Инструменты |
|-----------|-------------|
| **Язык & Скриптинг** | Bash, awk, grep, sed, cron |
| **Логи & Анализ** | Nginx combined log format, GoAccess, текстовые логи |
| **Мониторинг** | Prometheus, Grafana, node_exporter, custom bash-exporter |
| **Нагрузка & Тесты** | stress, iperf3, dd/bs (генерация файлов) |
| **ОС & Инфра** | Ubuntu Server 20.04 LTS, Nginx, VirtualBox |
