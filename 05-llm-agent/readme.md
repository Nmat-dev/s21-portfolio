# 🤖 ИИ-агент

Автономный ассистент на базе LLM (GigaChat) для поддержки образовательного пути. Воркфлоу реализованы в n8n, интерфейс — Telegram-бот. Проект охватывает промпт-инжиниринг, интеграцию внешних API, управление контекстом и базовую реализацию RAG.

## 🎯 Цель проекта
Освоить создание агентных систем на базе LLM: от настройки инфраструктуры до разработки проактивных сценариев, работы с памятью, инструментами (tools) и векторным поиском.

## 🛠 Технологический стек
| Категория | Инструменты |
|-----------|-------------|
| **Оркестрация** | n8n (self-hosted), Docker, Cloud.ru Container Apps |
| **LLM & AI** | GigaChat API (модели 2.0), промпт-инжиниринг, System/User messages |
| **Интеграции** | Telegram Bot API, Tavily Search, HH.ru API, RSS-парсинг (FetchRSS) |
| **Память & Данные** | SimpleMemory, Static Data (n8n), Simple Vector Store, Hugging Face embeddings (ru-en-RoSBERTa) |
| **Инфраструктура** | Ubuntu Server, Nginx (reverse proxy), cron, bash-скрипты для деплоя |
| **Безопасность** | Анонимизация данных, управление API-ключами, .env-конфигурация |

## ✅ Что реализовано
### 🔧 Часть 1: Настройка инфраструктуры
Развёртывание n8n в Cloud.ru через Container Apps + Docker
Подключение GigaChat API через узел OpenAI Chat Model (с отключением Responses API)
Базовый воркфлоу: Telegram-триггер → LLM → ответ пользователю
### 🎯 Часть 2: Агент-карьерный консультант
Системный промпт с ролью, целью и правилами поведения («Ты карьерный консультант...»)
Серия вопросов: опыт, интересы, цели, сильные стороны, предпочтения по работе
Интеграция с данными отборочного бассейна (анонимизированный анализ метрик)
Пространство памяти: SimpleMemory для хранения контекста диалога
Финальная рекомендация по специализации с обоснованием
### 📰 Часть 3: Дайджест новостей
Парсинг 3+ Telegram-каналов через RSS-прослойку (FetchRSS)
Двухэтапная суммаризация: пост → абзац → общий дайджест
Ежедневная отправка в приватный канал с форматированием и ссылками
Контроль дубликатов через хранение обработанных постов
### 💼 Часть 4: Анализ рынка вакансий
Интеграция с HH.ru API: поиск вакансий по направлению и региону
Извлечение навыков через GigaChat + нормализация («python» / «питон» → «python»)
Агрегация топ-15 требований с подсчётом частоты
Еженедельный отчёт в Telegram с интерпретацией трендов
Осознанная работа с рисками: стохастичность LLM, анонимизация данных
### 🎓 Часть 5: Персональный ментор
Интерактивный сбор запроса: тема → уровень → цели
Генерация контент-плана: темы → понятия → последовательность
Интеграция с Tavily Search: поиск актуальных материалов по понятиям
Обход ограничения tool_calls в GigaChat 2: кастомная цепочка (AI Agent → JS-проверка → HTTP Request → AI Agent)
Хранение прогресса: Static Data для отслеживания пройденных тем
Отправка постов без повторов, запрос следующей темы после завершения
### 🧠 Часть 6: Бонус — базовый RAG
Загрузка PDF-документов (вакансии, резюме) через n8n File-узлы
Чанкинг текста + эмбеддинги через Hugging Face (ru-en-RoSBERTa)
Векторное хранилище: Simple Vector Store для поиска релевантных фрагментов
Генерация ответа на основе найденных чанков + GigaChat 2
Оценка качества: сравнение ответа с ожиданиями, итеративная настройка промптов
### 🧪 Как воспроизвести (кратко)
Зарегистрироваться на Cloud.ru, активировать промокод
Развернуть n8n через Container Apps (следовать инструкции)
Создать Telegram-бота через @BotFather, получить токен
Настроить GigaChat API: создать ключ, добавить в n8n Credentials
Импортировать воркфлоу из src/0X_*/workflow.json в n8n
Заполнить переменные окружения (токены, API-ключи, Chat ID)
Активировать воркфлоу и протестировать через Telegram
