# 🐧 Операционные системы UNIX/Linux (Базовый)
> Первый проект трека DevOps. Установка, базовое администрирование и настройка Ubuntu Server 20.04 LTS.

## ✅ Что реализовано
- Установка Ubuntu Server без GUI в VirtualBox
- Создание пользователей, настройка `sudo`, hostname, временной зоны, NTP-синхронизация
- Статическая настройка сети (IP, GW, DNS), проверка связи (`ping`, `traceroute`)
- Работа с текстовыми редакторами (VIM, Nano, Joe)
- Настройка SSHd на нестандартном порту (2022), анализ логов аутентификации
- Мониторинг ресурсов через `top`/`htop`, анализ дисков (`fdisk`, `df`, `du`, `ncdu`)
- Планирование задач через `cron`, работа с системными журналами (`dmesg`, `syslog`, `auth.log`)

## 🛠 Стек
`Ubuntu 20.04 LTS` `VirtualBox` `Bash` `systemd` `cron` `SSH` `VIM` `netplan`
