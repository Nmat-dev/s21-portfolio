# 🌐 Сети в Linux

> Практический проект по настройке и администрированию сетей на базе Ubuntu Server 20.04 LTS. Развёрнута лабораторная инфраструктура из 5 виртуальных машин с маршрутизацией, DHCP, NAT и фаерволами.

## 🎯 Цель проекта
Освоить фундаментальные принципы работы сетей в Linux: адресация, маршрутизация, фильтрация трафика, сервисы (DHCP, NAT) и диагностика.

## 🛠 Технологический стек
| Категория | Инструменты |
|-----------|-------------|
| **ОС & Виртуализация** | Ubuntu Server 20.04 LTS, VirtualBox |
| **Сетевая конфигурация** | `netplan`, `ip`, `ipcalc`, `sysctl` |
| **Маршрутизация** | Статические маршруты, `ip r`, таблица маршрутизации |
| **Диагностика** | `ping`, `tcpdump`, `traceroute`, `iperf3`, `nmap`, `netstat` |
| **Безопасность** | `iptables` (filter, NAT), стратегии разрешающих/запрещающих правил |
| **Сервисы** | `isc-dhcp-server`, Apache2, SSH, SSH Tunnels (Local/Remote forwarding) |
| **Протоколы** | TCP/UDP, IP, ICMP, DHCP, NAT (SNAT/DNAT), HTTP |
